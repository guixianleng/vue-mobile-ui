### Cell 单元格

<cell />

### API
|       参数      |         说明        |    类型  |      默认值    |
|-----------------|---------------------|----------|----------------|
| value           | 展开的内容                     | `String`       | -       |
| type            | 展开方式, 可选值`character`    | `String`       | `arrow` |
| align           | 展开动作的位置`left` `right`   | `String`       | `center`|
| unfold-text     | 展开描述文字                   | `String`       | `展开`  |
| fold-text       | 折叠描述文字                   | `String`       | `收起`  |
