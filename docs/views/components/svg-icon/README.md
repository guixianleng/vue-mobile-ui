### Svg-icon

#### API

|       参数      |         说明        |    类型  |      默认值    |
|-----------------|---------------------|----------|----------------|
| icon            | svg地址                         | `String`      | -      |
| size            | 图标大小，如 20px 2em，默认单位为px | `String`  | `188`  |
| animation       | 是否开启动画                    | `Boolean`     | `false`|
| animation-name  | 是否开启动画名                  | `String`      | -      |
